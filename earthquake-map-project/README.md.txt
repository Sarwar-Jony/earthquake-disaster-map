# Deadliest Earthquakes Map (2000-2020)

[cite_start]This is a cartographic project visualizing 10 of the deadliest earthquakes that occurred between the years 2000 and 2020. [cite_start]The map uses proportional symbols to represent the number of fatalities[cite: 6], providing a stark visual of the human cost of these events.

![Earthquake Map](10_Significant_Earthquakes_2000-2020.png)

## Project Details

This map was created to highlight the geographic distribution and human impact of major seismic events. [cite_start]It includes key data such as the location of the earthquake and its associated death toll, such as the 87,652 deaths in China's Sichuan Province[cite: 28, 29].

### Features
* [cite_start]**Proportional Symbols:** Circle sizes correspond to the total number of deaths[cite: 6].
* [cite_start]**Tectonic Context:** Major fault lines are included to show the relationship between plate boundaries and seismic activity.
* **Clear Labeling:** Each of the 10 significant events is clearly labeled with location and fatality data.

### Tools and Data
* [cite_start]**Software:** Created with QGIS.
* [cite_start]**Data Sources:** Natural Earth, GEMS Fault Database, and NECI Earthquake Database[cite: 21, 22, 23].
