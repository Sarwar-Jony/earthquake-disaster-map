# 🌍 10 Largest Earthquakes (2000–2020)

A professional GIS and Cartography project visualizing the **10 deadliest earthquakes between 2000 and 2020** using **QGIS**. The map presents earthquake locations, fault lines, and the total number of deaths through proportional symbols and informative labels.

![Project Preview](images/10_Largest_Earthquake_2000_2020.png)

---

## 📌 Project Overview

This project demonstrates GIS-based thematic mapping techniques by combining earthquake event data with global tectonic fault information.

The primary objective is to create a clean, publication-quality map that effectively communicates the spatial distribution and impact of major earthquakes around the world.
